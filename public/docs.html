API Documentation

KeyAuth 1.3 Client API

Base URL: /api/1.3/

Compatible with standard KeyAuth client libraries. Change the URL from https://keyauth.win/api/1.3/ to your Vercel URL.

Client Endpoints

TypeDescriptionRequired Params
initInitialize sessionname, ownerid
registerRegister with license keysessionid, username, pass, key
loginLogin with credentialssessionid, username, pass
licenseLogin with license key onlysessionid, key
logoutEnd sessionsessionid
checkVerify active sessionsessionid
upgradeRedeem license on accountsessionid, username, key
varGet global variablesessionid, varid
getvarGet user variablesessionid, var
setvarSet user variablesessionid, var, data
chatgetGet chat messagessessionid
chatsendSend chat messagesessionid, message

Seller API

Base: /api/seller?action=ACTION&ownerid=ID&secret=SECRET

ActionDescription
createlicenseGenerate keys (amount, duration, durationUnit, note)
fetchlicensesList all licenses
fetchusersList all users
fetchstatsGet app statistics
banuser / unbanuserBan or unban a user
setvar / fetchvarsManage global variables

Example: Initialize + License Login

GET /api/1.3/?type=init&name=MyApp&ownerid=YOUR_OWNER_ID&ver=1.0
→ { "success": true, "sessionid": "abc123..." }

GET /api/1.3/?type=license&sessionid=abc123&key=XXXXX-XXXXX-XXXXX-XXXXX&hwid=machine-id
→ { "success": true, "info": { "username": "user_XXXXX", "subscriptions": [...] } }
